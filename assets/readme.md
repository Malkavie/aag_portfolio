DESCRIPTION:
This project is a simulated industrial monitoring system designed to track production flow in real time.

It models a manufacturing pipeline where each unit moves through multiple stages, with dynamic state updates, error handling, and live metrics.

The system was built using vanilla JavaScript and SVG to demonstrate full control over rendering, animation, and state management without external libraries.

KEY FEATURES:
Real-time state simulation and updates
Event-driven logic (normal flow vs error handling)
Dynamic UI updates and system feedback
Interactive controls (pause/resume, manual error injection)
Live metrics (processed units, error rate, system status)
Bilingual UI (English / Spanish)

This project reflects real-world scenarios where monitoring tools are used to visualize production, detect issues, and support operational decision-making.
